# express-policy-guard

Declarative, context-aware authorization middleware for Express.js.

`express-policy-guard` helps you define **clean, testable, and explainable**
authorization rules for your Express applications â€” without coupling your
business logic to routes.

---

## Features

- Express-first authorization middleware
- Declarative policy definitions
- Context-aware access control
- Async policy support
- Inline route-level conditions
- Explainable deny reasons
- TypeScript ready
- Fully tested with Jest & Supertest

---

## Installation

```bash
npm install express-policy-guard
```
